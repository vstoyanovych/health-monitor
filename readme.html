SiMan CMS

Installation manual
Copy Siman CMS files to your site directory.

You need to set values of hostNameDB, nameDB, userNameDB, userPasswordDB (MySQL server name, database name, database user name, database user password) in the file includes/dbsettings.php.

Set write access mode (777) to directory files in CMS directory.

Run install/install.php file (http://yoursite/path_to_siman/install/install.php or http://yoursite/install/install.php) in your browser.

Erase directory install from CMS directory after successfull installation!


See more details at official site simancms.apserver.org.ua.
